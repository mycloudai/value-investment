# 使用帮助 · MyCloudAI 巴菲特的价值投资

> 本站是一个巴菲特价值投资学习平台，收录了 1957–2024 年间的合伙人信、股东信、特别信件，以及系统整理的投资概念、公司分析和关键人物介绍。

## 🔍 搜索功能

- **唤出搜索**：点击顶部「🔍 搜索」按钮，或使用快捷键：
  - **Mac**：`⌘ Cmd + K`
  - **Windows / Linux**：`Ctrl + K`
- 搜索框自动聚焦，实时显示结果（段落级别全文搜索）
- 每条结果显示：文章标题 + 命中段落摘要 + 分类标签
- **键盘导航**：`↑ / ↓` 切换结果，`Enter` 跳转，`ESC` 关闭
- 点击结果 → **新标签页**打开对应文章，并自动**滚动到命中段落**（无关键词高亮）
- 搜索支持中英文混合
- **在某篇信件中搜索**：在概念/公司/人物页点击关联信件，弹出该信件的范围内搜索

## 📑 文章阅读

### 右侧面板

文章页面右侧会显示两类信息：

**📋 目录（TOC）**
- 自动从文章标题生成
- 点击目录条目 → 平滑滚动到对应段落
- 仅在文章有 2 个以上标题时显示

**🔗 关联内容**

根据页面类型不同，行为有所区别：

| 当前页面 | 关联内容显示 | 点击行为 |
|---------|------------|---------|
| **信件页**（股东信/合伙人信/特别信件）| 本信涉及的概念、公司、人物 | 新标签页打开对应概念/公司/人物页面 |
| **概念/公司/人物页** | 哪些信件提到了此内容 | 调用搜索，在该信件中定位相关段落，可选择跳转 |

## 📝 金句收藏

- 侧边栏点击「**金句**」查看所有巴菲特精选语录（483 条）
- 每篇文章底部也会展示与本文相关的金句
- 金句支持按分类筛选（全部 / 股东信 / 合伙人信 / 概念 / 公司 / 人物）

> **开发者注意：** 金句的唯一来源是 `content/{category}/{slug}-quotes.md` 文件（153 个）。
> `data/quotes.yml` 和 `data/scraped-quotes.yml` 是已归档的历史文件，请勿编辑。
> 构建时 `build.mjs` 自动从 `*-quotes.md` 编译生成 `site/assets/data/quotes-data.json`。

## 🤖 AI 问答

1. 点击侧边栏「**AI问答**」
2. 在设置中填入 API Key（支持 OpenAI 和 Claude 格式）
3. 可以点击「获取模型列表」自动拉取可用模型
4. AI 以巴菲特视角回答，引用真实信件原文
5. 支持工具调用：AI 会主动搜索知识库获取准确信息

**支持的 API 格式：**
- **OpenAI 格式**：`https://api.openai.com/v1`（或任何兼容接口）
- **Claude 格式**：`https://api.anthropic.com`

## 📐 侧边栏折叠

- 侧边栏底部有「**点击 - 收起导航**」按钮，点击后侧边栏向左收起，主内容区自动展开
- 折叠后，页面左侧边缘出现竖排蓝色「**导航**」标签，点击即可重新展开侧边栏
- 折叠状态会自动保存到本地，刷新页面后保持上次的状态
- 在移动端，折叠逻辑由顶部汉堡菜单控制，桌面端折叠按钮不显示

## 🌐 双语阅读

每封信件（股东信、合伙人信、特别信件）均提供中英双语切换：

- **中文** — 默认显示中文翻译全文
- **English** — 切换为英文原文，可单击任意英文单词查看中文释义
- **对照** — 左右并排显示中英文，适合对照学习

### 英文单词翻译

在 English 或对照模式下，单击任意英文单词弹出翻译气泡：
- 显示中文释义（由 MyMemory 翻译服务提供）
- 点击 🔊 按钮可朗读单词发音
- 点击 ✕ 或气泡外任意位置关闭

> 语言偏好按文章保存，下次打开同一封信会自动恢复。
